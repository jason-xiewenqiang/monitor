## 快速启动 | Quick Start

```
# 克隆项目
git clone https://github.com/GeekQiaQia/vue3.0-template-admin.git

# 进入项目目录
cd vue3.0-template-admin

# 安装依赖
npm install

# 启动服务
npm run dev

```
